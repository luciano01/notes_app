## For developers
> To clone locally
```
$ git clone git@github.com:luciano01/notes_app.git
$ cd notes_app
```
> Open the project in an editor, if you use VS Code, follow this
```
$ cd notes_app
$ code .
```
> To run, follow this on terminal
```
$ flutter clean
$ flutter pub get
$ flutter run --release
```
